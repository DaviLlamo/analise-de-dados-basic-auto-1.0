# analise-de-dados-basic-auto-1.0
vers.1.0 de uma automatização que pega um arquivo de excel e transforma em grafico mais facilmente visivel e rapido caso o excel não tenha

##usei:
- Python
- Pandas
- Matplotlib
- Tkinter
**se caso for usar esse o codigo tera que instalar essas bibliotecas do python com o comando no terminal pip install -r requirements.txt**

## Como usar

1. Execute o programa


python main.py


2. Clique em **Selecionar Excel**

3. Escolha um arquivo Excel

O programa irá gerar gráficos automaticamente com base nas colunas numéricas.

## Exemplo de Excel suportado

| Nome | Jan | Fev | Mar |
|-----|----|----|----|
| Ana | 10 | 12 | 8 |
| João | 5 | 7 | 9 |
| Maria | 3 | 6 | 4 |
